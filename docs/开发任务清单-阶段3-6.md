- [ ] 实现 SAR 汇总计算
  - 合计 SAR = 省大区 SAR + 网络经销商 SAR + 海外出口 SAR + 内部调拨需求
- [ ] 实现总供应计算
  - 总供应 = 期初库存 + 排产计划 - 安全库存预留
- [ ] 实现缺口计算
  - 缺口 = 总供应 - 合计 SAR
- [ ] 实现满足率计算
  - 满足率 = 可满足合计 / 合计 SAR
- [ ] 使用 pandas 向量化计算提升性能

**验收标准**：
- 计算结果准确
- 5000 条 SKU 计算时间 < 5 秒
