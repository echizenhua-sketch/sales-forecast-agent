### 1. SAR 字段完全对齐（P0 关键修正）

#### ❌ 修正前

```sql
-- 只有 4 个 SAR 字段，命名错误
sar_province DECIMAL(18, 2) COMMENT '省大区SAR',
sar_dealer DECIMAL(18, 2) COMMENT '网络经销商SAR',
sar_export DECIMAL(18, 2) COMMENT '海外出口SAR',      -- ❌ 应为"电商直营SAR"
sar_internal DECIMAL(18, 2) COMMENT '内部调拨需求',   -- ❌ 应为"KA部SAR"
-- ❌ 缺少"拓展部SAR"
sar_total DECIMAL(18, 2) COMMENT '合计SAR',
```

**问题**：
- 与 PRD v2.1 第 493 行公式不符
- Excel 模板有 5 个 SAR 字段，数据库只有 4 个

#### ✅ 修正后

```sql
-- 5 个 SAR 字段完整，与 PRD v2.1 完全一致
sar_province DECIMAL(18, 2) COMMENT '省大区SAR',
sar_dealer DECIMAL(18, 2) COMMENT '网络经销商SAR',
sar_ecommerce DECIMAL(18, 2) COMMENT '电商直营SAR',    -- ✅ 修正
sar_ka DECIMAL(18, 2) COMMENT 'KA部SAR',               -- ✅ 修正
sar_expansion DECIMAL(18, 2) COMMENT '拓展部SAR',      -- ✅ 新增
sar_total DECIMAL(18, 2) COMMENT '合计SAR (= 省大区 + 网络经销商 + 电商直营 + KA部 + 拓展部)',
```

**公式验证**：
```
PRD v2.1: 5月合计SAR = 省大区SAR + 网络经销商SAR + 电商直营SAR + KA部SAR + 拓展部SAR
v2.2:     sar_total = sar_province + sar_dealer + sar_ecommerce + sar_ka + sar_expansion
```
✅ **完全一致**
